验证码请求

  • 接口地址： /captcha/get
  • 请求方式： http POST
  • 返回格式： json
  • 接口备注： 每次请求需要生成一个随机id（如当前时间的hash值），用于请求的标识。
  • 请求方式说明：
    名称 必填 类型 说明
      id 是 string 不超过32位的随机字符串，用于唯一标识当前请求。
      user 否 string 当前请求的用户名，用于统计。

验证码验证

  • 接口地址： /captcha/check
  • 请求方式： http POST
  • 返回格式： json
  • 请求方式说明：
    名称 必填 类型 说明
      id 是 string 不超过32位的字符串，对应请求时的id。
      ans 是 string 当前检查的答案。
